  <name>Task 1: Implement AudioSession (start/stop, mix/resample, stream to disk)</name>
  <files>
src/metamemory/audio/__init__.py
src/metamemory/audio/session.py
  </files>
  <action>
- Create `src/metamemory/audio/session.py` exposing an `AudioSession` with:
  - `start(config) -> None` and `stop() -> Path` (returns finalized WAV path).
  - Config includes: selected sources (mic/system/fake), device ids (optional), output dir override (for tests), and gains.
- Session responsibilities:
  - Start selected sources from `metamemory.audio.capture`.
  - Run a consumer thread that reads from each source queue, converts to float32 numpy, downmixes to mono, resamples each to 16000Hz via `soxr`, mixes, converts to int16, and writes bytes to `PcmPartWriter`.
  - Ensure bounded memory: queues must be bounded; on overflow, drop frames and record a counter.
  - On stop: stop sources, drain remaining frames briefly, close writer, finalize to WAV, return path.
- Keep PyQt out of this module (pure Python) so it can be tested.
  </action>
  <verify>python -c "from metamemory.audio.session import AudioSession; print(AudioSession)"</verify>
  <done>
- Session API is stable and independent of UI.
- Output is always 16kHz/16-bit/mono.
  </done>
